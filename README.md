> Backend Chat bot IA

1. Create Virtual Enviroment

```
python -m venv venv
```

2. Activate enviroment

```sh
source venv/bin/activate
 -or-
source venv/Scripts/activate
```

3. Install dependecies

```sh
pip install -r api/requirements.txt
```

4. start the server
   inside folder -> api/

```sh
uvicorn main:app --reload
```
